Voici le TP-Manga-Entity de Louis PONT.

Pour Axopen : Je n'ai pas codé le projet de A à Z, tous les élèves sont partis d'une base donnée par le professeur.

Voici les fonctionnalités que j'ai implémentées :

La connexion d'un utilisateur (avec vérification du mot de passe)
La déconnexion d'un utilisateur
La création d’un compte utilisateur (il sera en lecture)
L’ajout d’un dessinateur/scénariste/style et manga
La modification d’un manga
La suppression d’un manga
La recherche de mangas à l’aide d’un critère
L’augmentation du prix grâce à la procédure stockée
Le visionnage d’un manga (codé en dur dans le code suite à la mise à jour du connecteur entrainant un problème)
